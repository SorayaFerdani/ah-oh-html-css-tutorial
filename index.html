HTML (HyperText Markup Language)

Dient dazu die Website zu strukturieren

Besteht aus verschiedenen Elementen (Tags)

Tags

Die meisten Elemente haben einen öffnenden und schließenden Tag

- öffnende Tag: Namen des Elements zwischen zwei spitzen Klammern

- Inhalt: Inhalt des Elements

- schließende Tag: genauso wie das öffnende Tag bis auf den zusätzlichen Schrägstrich

- Das Element: Das öffnende Tag, der Inhalt und das schließende Tag gemeinsam
